# تطبيق مؤذني - Moatheny

## نظرة عامة

تطبيق إسلامي شامل يوفر:
- 🕌 **أوقات الصلاة** مع أذان تلقائي
- 📖 **القرآن الكريم** كاملاً مع التلاوات الصوتية
- 📿 **الأذكار** منظمة حسب الفئات
- 🧭 **اتجاه القبلة** بدقة 100%
- 📱 **Widget** لشاشة القفل والشاشة الرئيسية

---

## المزايا

### 1. أوقات الصلاة

- **الحساب التلقائي**: يحسب أوقات الصلاة بناءً على موقعك GPS
- **طرق حساب متعددة**:
  - رابطة العالم الإسلامي
  - الهيئة المصرية العامة للمساحة
  - جامعة أم القرى (السعودية)
  - الجمعية الإسلامية لأمريكا الشمالية
  - جامعة العلوم الإسلامية، كراتشي
- **إشعارات الأذان**: تنبيهات صوتية عند كل صلاة
- **تذكير قبل الأذان**: 5، 10، 15، 30 دقيقة قبل

### 2. القرآن الكريم

- **النص الكامل**: 114 سورة بالرسم العثماني
- **التلاوات الصوتية**: 
  - مشاري العفاسي
  - عبد الرحمن السديس
  - سعود الشريم
  - ماهر المعيقلي
  - وأكثر من 50 قارئ
- **الترجمات**: متوفرة بأكثر من 40 لغة
- **البحث**: بحث سريع في السور والآيات
- **التحميل**: تنزيل التلاوات للاستماع بدون إنترنت

### 3. الأذكار

**الفئات المتوفرة:**
- أذكار الصباح (30+ ذكر)
- أذكار المساء (30+ ذكر)
- أذكار النوم
- أذكار الاستيقاظ
- أذكار بعد الصلاة
- أذكار الطعام
- أذكار السفر
- أذكار المسجد
- أذكار الوضوء
- أدعية قرآنية
- أدعية نبوية
- الرقية الشرعية
- أسماء الله الحسنى (99 اسم)

**المزايا:**
- عداد التكرار لكل ذكر
- تذكير يومي صباحاً ومساءً
- إمكانية الاستماع للأذكار

### 4. اتجاه القبلة

- **دقة 100%**: يحسب الاتجاه بدقة عالية باستخدام موقعك GPS
- **بوصلة ذكية محسنة**: تستخدم تقنيات متقدمة للدقة والاستقرار:
  - **Extended Kalman Filter**: تنعيم متقدم لقراءات البوصلة
  - **Magnetic Anomaly Detector**: كشف تلقائي للتشويش المغناطيسي
  - **Magnetic Declination Calculator**: تعويض الانحراف المغناطيسي
  - **Sensor Fusion Engine**: دمج Accelerometer, Gyroscope, Magnetometer
  - **Performance Metrics Collector**: مراقبة الأداء والتحسين
  - **Adaptive Update Rate Manager**: توفير البطارية مع الحفاظ على الدقة
- **بوصلة مرئية**: واجهة جميلة مع سهم يشير للقبلة
- **معلومات إضافية**:
  - المسافة إلى مكة بالكيلومتر
  - وصف الاتجاه (شمال، جنوب شرق، إلخ)
  - اسم المدينة والدولة
  - آخر وقت تحديث
- **يعمل في أي مكان في العالم**

### 5. المسبحة الإلكترونية

- عداد رقمي للتسبيح
- إمكانية إعادة العداد
- واجهة بسيطة وسهلة الاستخدام

## التقنيات المستخدمة

- **اللغة**: Swift 5.0 / SwiftUI
- **الإصدار المطلوب**: iOS 16+
- **المكتبات**:
  - `Adhan` - لحساب أوقات الصلاة
  - `SwiftSoup` - لاستخراج البيانات من المواقع
  - `AVFoundation` - لتشغيل الصوتيات
  - `CoreLocation` - للموقع الجغرافي
  - `CoreMotion` - للمستشعرات (Accelerometer, Gyroscope, Magnetometer)
  - `WidgetKit` - للويدجت
- **خوارزميات البوصلة**:
  - Extended Kalman Filter (EKF) - للتنعيم والدمج
  - World Magnetic Model (WMM) - لحساب الانحراف المغناطيسي
  - Statistical Anomaly Detection - لكشف التشويش
  - Sensor Fusion - لدمج المستشعرات المتعددة

## التثبيت والتشغيل

1. افتح المشروع في Xcode 15+
2. أضف Swift Packages:
   - `https://github.com/batoulapps/adhan-swift.git`
   - `https://github.com/scinfu/SwiftSoup.git`
3. اختر جهاز محاكاة أو جهاز حقيقي (iOS 16+)
4. اضغط Run (⌘R)
5. عند أول تشغيل:
   - اسمح بالوصول للموقع
   - اسمح بالإشعارات
6. التطبيق جاهز للاستخدام!

## الاستخدام

### أوقات الصلاة:
1. افتح تبويب "الصلاة"
2. امنح إذن الموقع عند الطلب
3. سيتم تحميل أوقات الصلاة تلقائياً
4. اضغط على أيقونة التحديث لتحديث الأوقات

### القرآن الكريم:
1. افتح تبويب "القرآن"
2. انتظر تحميل السور (يتطلب إنترنت للتحميل الأول)
3. اضغط على أي سورة لقراءة الآيات
4. (قريباً) استمع للتلاوة من قائمة القراء

### الأذكار:
1. افتح تبويب "الأذكار"
2. اختر الفئة المناسبة
3. اقرأ الأذكار والتكرار المطلوب
4. (قريباً) تفعيل التذكير اليومي

### اتجاه القبلة:
1. افتح تبويب "المزيد"
2. اختر "اتجاه القبلة"
3. امنح إذن الموقع إذا لم تفعل
4. انتظر حساب الاتجاه
5. اتجه نحو السهم الأخضر
6. اضغط "تحديث" إذا تغير موقعك
