## التحليل الرياضي

### الحساب اليدوي للرياض

**المعطيات:**
- موقع المستخدم: `φ₁ = 24.7136°N, λ₁ = 46.6753°E`
- موقع الكعبة (API): `φ₂ = 21.4224779°N, λ₂ = 39.8251832°E`

**الخطوات:**

1. **تحويل إلى راديان:**
   ```
   φ₁ = 24.7136 × π/180 = 0.4313 راديان
   λ₁ = 46.6753 × π/180 = 0.8146 راديان
   φ₂ = 21.4224779 × π/180 = 0.3739 راديان
   λ₂ = 39.8251832 × π/180 = 0.6950 راديان
   ```

2. **حساب فرق خطوط الطول:**
   ```
   Δλ = λ₂ - λ₁ = 0.6950 - 0.8146 = -0.1196 راديان
   ```

3. **تطبيق معادلة Great Circle Bearing:**
   ```
   y = sin(Δλ) × cos(φ₂)
     = sin(-0.1196) × cos(0.3739)
     = -0.1193 × 0.9308
     = -0.1111
   
   x = cos(φ₁) × sin(φ₂) - sin(φ₁) × cos(φ₂) × cos(Δλ)
     = cos(0.4313) × sin(0.3739) - sin(0.4313) × cos(0.3739) × cos(-0.1196)
     = 0.9084 × 0.3648 - 0.4181 × 0.9308 × 0.9929
     = 0.3314 - 0.3869
     = -0.0555
   
   bearing = atan2(y, x)
           = atan2(-0.1111, -0.0555)
           = -2.0344 راديان
           = -116.56°
   ```

4. **تطبيع إلى [0, 360]:**
   ```
   bearing = -116.56 + 360 = 243.44°
   ```

**النتيجة المحلية المتوقعة:** `243.44°`
